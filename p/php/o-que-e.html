O que é PHP?

É uma linguagem de script/programação utilizada para criar páginas dinâmicas na web.

Assim como o youtube na imagem acima, muitas outras páginas da web requerem que grande parte do seu conteúdo seja variável, isto é, seja diferente a depender de um certo parâmetro, no caso do youtube, os campos de inscrições, vídeos, canal, notificações e até mesmo anúncios são diferentes para cada usuário, baseado em suas preferências e região geográfica e para isso é necessário que se tenha uma forma de carregar esses dados de algum lugar, tratá-los e exibí-los em html/css. Existem muitas formas de fazer isso e o PHP é uma das mais populares delas.


O código do PHP é executado pelo computador servidor, o computador cliente/usuário apenas recebe o código html gerado pelo processamento do PHP.
