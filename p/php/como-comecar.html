Como começar?

Para começar com php é necessário ter um servidor com suporte ao php, uma forma de ter um no ambiente local(pelo localhost) é através do aplicativo wamp no windows.


⚪Wamp

Download WAMP

Basta fazer o download do wamp e instalá-lo com o procedimento normal de instalação.


Após instalar o wamp execute-o, um icone do wamp deve aparecer no painel de mini-icones da barra do windows.


Clique no icone do wamp e depois na opção Start All Services e pronto, o wamp deverá estar preparado para receber o php.


Clicando novamente no ícone do wamp na barra, é possível observar uma opção chamada www directory, clicando nela, uma pasta é aberta com o nome www, essa é a pasta raiz do servidor, ou seja, é aqui que o servidor busca a primeira página index.php para executar e é dentro dela que ficam as outras pastas de outras páginas que sejam relativas ao site.

O servidor começa a procurar os arquivos de site a partir da pasta www, no caso de um servidor local o endereço de acesso a essa pasta no navegador é localhost, mas em sites publicados é o próprio endereço do site, no caso do yahoo será o yahoo.com.br.


Se tivéssemos um arquivo na pasta www nomeado peixes.php o acessariámos simplesmente digitando localhost/peixes.php, caso não digitemos nada, deixemos apenas localhost, o servidor irá por pádrão tentar carregar o arquivo index.php se ele existir.


Para acessar arquivos em pastas, segue-se o mesmo raciocínio, para acessar a pasta contato, colocariamos localhost/contatos/algumarquivo.php ou ainda apenas localhost/contatos/ caso queira acessar o index.php dentro de contatos de uma forma mais curta.


Apenas reforçando, quando um site é publicado, a única coisa que muda nessa lógica é a palavra localhost, ela muda para por exemplo uol.com.br ou weshare.com, que são domínios, como exemplo, acessar a página contatos com o domínio da uol seguiria o mesmo exemplo do localhost, ficando uol.com.br/contatos, todas as urls são formadas por essa parte inicial que indica que página se quer acessar.
